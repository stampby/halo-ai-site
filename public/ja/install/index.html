インストール

halo-aiは2つのコマンドでインストールできる。コンテナなし、クラウド依存なし、テレメトリなし。すべてあなたのマシンで動く。

要件

コンポーネント 最低 推奨
RAM 16 GB 64 GB+
GPU AMDまたはNVIDIA（Vulkan対応） AMD Radeon 8060S以上
ストレージ 20 GB空き 50 GB+（モデルが容量を使用）
OS Linux (systemd) Arch Linux / CachyOS

対応ディストリビューション

  • Arch Linux – ファーストクラスサポート、毎日テスト済み
  • CachyOS – Archベース、パフォーマンスチューニング済み、完全互換
  • Ubuntu 22.04+ – サポート対象
  • Fedora 39+ – サポート対象

クイックスタート

git clone https://github.com/halo-ai-official/halo-ai.git
cd halo-ai
./install.sh

インストーラーがハードウェアを検出し、GPUアクセラレーションを設定し、モデルをダウンロードし、33のサービスすべてを起動する。AMD Strix Haloハードウェアなら、完全なデプロイは10分以内に完了する。

インストールされるもの

halo-aiは推論、音声合成、音声認識、画像生成、コード分析、セキュリティ監視、自律エージェント連携をカバーする33の相互接続サービスをデプロイする。各サービスはsystemdで管理され、個別に起動・停止・検査が可能。

AMD Strix Halo最適化

インストーラーはStrix Haloシステム向けのハードウェア固有のチューニングを自動的に適用する。Vulkanバックエンドの選択、Flash Attentionの有効化、LPDDR5x向けメモリ割り当て戦略、ファンカーブプロファイルが含まれる。Strix Haloマシンで動かすなら、halo-aiはそれを見つけ、すべてのワットを使い切る。

インストール後

インストール完了後、ブラウザを開いてローカルのhalo-aiインスタンスにアクセスしよう。Man Caveダッシュボードが待っている。

トラブルシューティング、アーキテクチャの詳細、サービスドキュメントについてはドキュメントハブを参照。
